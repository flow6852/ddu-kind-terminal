# ddu-kind-terminal

Terminal kind for ddu.vim.

## Required

### denops.vim

https://github.com/vim-denops/denops.vim

### ddu.vim

https://github.com/Shougo/ddu.vim

### ddu-source-terminal

https://github.com/flow6852/ddu-source-terminal

## Configuration

```vim
	call ddu#start(#{ sources: [#{ name: 'terminal' }] })
```
